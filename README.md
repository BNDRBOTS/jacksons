'Jacksons Limousine'
Custom web build for Scottsdale and Phoenix private chauffeur operations. Designed for strict accuracy and uncompromised visual fidelity.

Visual Mechanics
OKLCH color matrix utilizing pure black and titanium contrasts. Hardware-native cursors. WebGL fragment shader calculating liquid depth and pointer tracking.

Motion
GSAP batched scroll triggers, magnetic button physics, and hardware-backed transitions.

Interaction
Slide-out reservation interface routing directly to dispatch. Accessible input checking with immediate success state handling.

Resource Management
IntersectionObserver throttling for GPU load management, localized DOM measurement caching, and strict reduced-motion fallback compliance.
